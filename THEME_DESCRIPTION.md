# Arcadia Theme - Obsidian Community Submission

## Short Description (for theme browser)
A comprehensive theme with 23 color schemes, OneNote-style features, 50+ file icons, 21 checkbox types, paper styles, and sticky notes. Fully customizable via Style Settings.

## Feature Highlights for Submission PR

### Color Schemes
- 11 dark themes: Granite, Stone, Olive, Flint, Navy, Ember, Twilight, Forest, Crimson, Slate, Amber
- 12 light themes: Marble, Sage, Pearl, Cloud, Dawn, Lavender, Mint, Rose, Silver, Cream, NY Times, Wikipedia

### Unique Features
- **50+ file type icons** with custom colors
- **21 alternate checkbox types** (?, !, *, ", l, i, b, S, p, c, /, u, d, w, k, f, I, etc.)
- **OneNote-style media embedding** with polished shadows, hover effects
- **Paper styles**: Lined, grid, dot, legal pad, Cornell notes, isometric, music staff
- **Sticky notes callouts** in 8 colors with pushpin/folded corner options
- **Code syntax themes**: Dracula, Nord, GitHub Dark/Light
- **Rainbow folders** for visual organization
- **Cards view** for Dataview integration
- **WCAG AA accessibility** compliance
- **OpenDyslexic font** support

### Style Settings Integration
60+ customization options including:
- Color scheme selection
- Typography (8+ fonts)
- 15 accent color choices
- Code block styling
- File icon toggle
- Paper style selection
- And more...
