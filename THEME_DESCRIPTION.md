# Arcadia Theme - Obsidian Community Submission

## Short Description (for theme browser)
A comprehensive theme with 34 color schemes (16 dark + 18 light), OneNote-style features, 50+ file icons, 21 checkbox types, paper styles, and sticky notes. Fully customizable via Style Settings.

## Feature Highlights for Submission PR

### Color Schemes
- 16 dark themes: Granite, Stone, Olive, Flint, Navy, Ember, Twilight, Forest, Crimson, Slate, Amber, Parchment, Heritage, Sepia, Registry, Rootwood
- 18 light themes: Marble, Sage, Pearl, Cloud, Dawn, Lavender, Mint, Rose, Silver, Cream, NY Times, Wikipedia, Crownwell, Parchment, Heritage, Sepia, Registry, Rootwood

### Unique Features
- **50+ file type icons** with custom colors
- **21 alternate checkbox types** (?, !, *, ", l, i, b, S, p, c, /, u, d, w, k, f, I, etc.)
- **OneNote-style media embedding** with polished shadows, hover effects
- **Paper styles**: Lined, grid, dot, legal pad, Cornell notes, graph, music staff
- **Sticky notes callouts** in 8 colors with pushpin/folded corner options
- **Code syntax themes**: Dracula, Nord, GitHub Dark/Light
- **Rainbow folders** for visual organization
- **Cards view** for Dataview integration
- **WCAG AA accessibility** compliance
- **OpenDyslexic font** support
- **14 content fonts** selectable via Style Settings

### Style Settings Integration
55+ customization options across 11 categories including:
- Color scheme selection (34 schemes)
- Typography (14 content fonts, 4 code fonts)
- 15 accent color choices
- Code block styling
- File icon toggle
- Paper style selection
- And more...
